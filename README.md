## 🚀 What this app does

write-like-me is a Rust app that learns how you write and turns that into a style skill for Claude Code.

You give it a sample of your writing. It studies your word choice, sentence length, tone, and rhythm. Then it builds a style model that Claude Code can use when it writes for you.

Use it when you want Claude Code to sound more like you and less like a generic assistant.

## ✍️ How it learns your writing

write-like-me builds a style profile from the text you give it.

It may look at:

- Sentence length
- Short vs long words
- Common phrases
- Tone
- Level of formality
- Punctuation use
- How often you use lists
- How you open and close ideas

For best results, use writing that sounds like you in real work. Good samples include:

- Emails
- Notes
- Blog posts
- Chat messages
- Documentation
- Drafts you wrote yourself

Use text that has enough detail. A few short lines will not give the tool much to work with.

## 🧩 How to use it with Claude Code

A normal flow looks like this:

1. Run write-like-me
2. Add one or more writing samples
3. Let it build your style model
4. Export the model or connect it to Claude Code
5. Ask Claude Code to write in your style

The tool acts like a writing profile that Claude Code can use while it drafts content. This helps keep the tone close to your own.

## ⚙️ Basic workflow

### Step 1: Add your writing

Paste or load text from files on your computer. Use text that is clean and easy to read. Avoid copied text from books or sites that do not sound like you.

### Step 2: Train the style model

The app scans your sample and records your habits. It may store the result in a local file so you can use it again later.

### Step 3: Connect the model

Point Claude Code to the saved style file or follow the app’s setup flow. If the tool gives you a path or prompt, copy it into Claude Code as shown.

### Step 4: Write with your style

Ask Claude Code to use your style profile when it drafts text. You can use it for messages, docs, notes, or code comments that need your voice.
